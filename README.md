# OGCard

Beautiful social media preview images in seconds. Free, no signup, no watermark.

## Features

- **8 Professional Templates**: Gradient Wave, Dark Elegant, Minimal Clean, Code Editor, Geometric, Sunset Gradient, Ocean Blue, Neon Glow
- **Live Preview**: Real-time canvas rendering as you type
- **Customizable**: Title, subtitle, author, accent color, font choice, logo upload
- **Multiple Sizes**: Twitter/X (1200x630), Facebook (1200x630), LinkedIn (1200x675), custom
- **HD Export**: Optional 2x resolution for crisp images
- **100% Client-Side**: No server, no data leaves your browser
- **Copy to Clipboard**: One-click copy for quick sharing

## Tech Stack

- Vite + Vanilla JS
- Tailwind CSS v4
- HTML5 Canvas API
- Google Fonts (Inter, Poppins, JetBrains Mono, Noto Sans TC)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/` — deploy as a static site.

## Deploy

```bash
npx vercel --yes
```
